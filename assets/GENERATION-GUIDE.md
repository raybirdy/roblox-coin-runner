# Coin Runner 에셋 생성 가이드 (Gemini 웹 사용)

gemini.google.com에서 아래 프롬프트를 순서대로 복사-붙여넣기하여 이미지를 생성하세요.
생성 후 다운로드하여 지정된 파일명으로 `assets/` 폴더에 저장하세요.

> **중요:** Gemini 웹에서는 비율 지정이 안 되므로, 프롬프트에 "가로로 넓은", "wide horizontal" 등을
> 명확히 포함했습니다. 정사각형이 필요한 이미지는 "정사각형 이미지" 문구를 포함합니다.

```
파스텔 초원 배경, 밝은 색감, 어린이 대상 게임.
정사각형, 텍스트 없음, 고대비.
```

```
오른쪽으로 장애물과 패럴랙스 배경 (언덕, 구름, 하늘)이 넓게 펼쳐짐.
밝고 에너지 넘치는 분위기, 아동 게임, 텍스트 없음.
가로로 긴 파노라마 구도, 절대 정사각형이 아닌 와이드 이미지.
```

## 저장 후 파일 구조

```
assets/
├── game-icon.png              # 512x512
├── game-thumbnail.png         # 1920x1080
├── GENERATION-GUIDE.md        # 이 파일
└── skins/
    ├── coin/
    │   ├── default.png        # 256x256
    │   ├── red_hat.png
    │   ├── ninja.png
    │   ├── blue_cape.png
    │   ├── pirate.png
    │   ├── robot.png
    │   └── golden.png
    └── premium/
        ├── galaxy.png         # 256x256
        ├── unicorn.png
        ├── ice_king.png
        ├── dragon.png
        ├── rainbow.png
        └── shadow.png
```

## 팁

- 스킨은 **일관된 스타일**을 위해 하나의 채팅 세션에서 연속 생성하세요
- 생성 후 마음에 안 들면 "더 밝게", "코인을 더 크게" 등 피드백을 추가해서 재생성
- 다운로드 시 위 파일명으로 저장하면 됩니다
