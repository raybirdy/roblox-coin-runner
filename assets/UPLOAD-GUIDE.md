# Coin Runner - Roblox Creator Hub 업로드 가이드

생성된 에셋을 Roblox Creator Hub에 업로드하고 게임에 적용하는 단계별 가이드입니다.

---

## 1단계: 게임 아이콘 & 썸네일 업로드

### 게임 아이콘 (512x512)

1. https://create.roblox.com 접속 → 로그인
2. **Creations** → **Coin Runner** 게임 선택
3. 좌측 메뉴에서 **Settings** → **Basic Info**
4. **Icon** 섹션에서 `assets/game-icon.png` 업로드
5. **Save** 클릭

### 게임 썸네일 (1920x1080)

1. 같은 Settings 페이지에서 **Thumbnails** 섹션
2. **Add Thumbnail** → `assets/game-thumbnail.png` 업로드
3. **Save** 클릭

> 썸네일은 최대 3장까지 등록 가능합니다. 추후 추가 썸네일을 만들 수 있습니다.

---

## 2단계: 스킨 이미지를 Decal로 업로드

스킨 이미지는 **Decal**로 업로드하여 게임 내 상점 UI에서 사용합니다.

### 업로드 절차 (13개 반복)

1. https://create.roblox.com 접속
2. 좌측 메뉴에서 **Development Items** → **Decals**
3. **Create a Decal** 클릭
4. 이미지 파일 업로드 + 이름 입력
5. **Upload** 클릭
6. 업로드 완료 후 **Asset ID** 메모

### 업로드 에셋 상세 정보

각 에셋별 **이름(Name)**, **설명(Description)**, **가격** 을 아래대로 입력하세요.

> Decal은 게임 내부 전용이므로 가격은 모두 **Off Sale (비매품)** 으로 설정합니다.

## 3단계: 코드에 Asset ID 적용

모든 Decal 업로드 후 메모한 Asset ID를 코드에 반영합니다.

### `src/shared/Constants/GameConstants.luau` 수정

현재 스킨 데이터에 `thumbnailId` 필드를 추가합니다:

```luau
-- 수정 전
{ id = "default", name = "기본", price = 0, currency = "coins", color = Color3.fromRGB(0, 170, 255) },

-- 수정 후
{ id = "default", name = "기본", price = 0, currency = "coins", color = Color3.fromRGB(0, 170, 255), thumbnailId = "rbxassetid://여기에_Asset_ID" },
```

## 4단계: 상점 UI에서 썸네일 표시 (코드 수정)

`ShopController`에서 `thumbnailId`가 있으면 색상 블록 대신 이미지를 표시하도록 수정합니다.

### 수정 위치

`src/client/UI/` 또는 `src/client/Controllers/` 내 상점 관련 파일에서
스킨 아이템 프레임을 생성하는 부분을 찾아 아래 로직을 추가합니다:

```luau
-- thumbnailId가 있으면 이미지 표시, 없으면 기존 색상 블록 유지
if skinData.thumbnailId then
    local imageLabel = Instance.new("ImageLabel")
    imageLabel.Image = skinData.thumbnailId
    imageLabel.Size = UDim2.new(1, 0, 1, 0)
    imageLabel.BackgroundTransparency = 1
    imageLabel.ScaleType = Enum.ScaleType.Fit
    imageLabel.Parent = itemFrame
else
    -- 기존 색상 블록 코드 유지
end
```

## 주의사항

- Roblox 심사(Moderation)를 통과해야 이미지가 표시됩니다 (보통 수 분 ~ 수 시간)
- 심사 거부 시: 부적절한 콘텐츠 포함 여부 확인 후 재업로드
- Decal 업로드 시 **Roblox 커뮤니티 기준** 준수 필수
- Asset ID는 숫자만 메모하고, 코드에서는 `"rbxassetid://숫자"` 형태로 사용
