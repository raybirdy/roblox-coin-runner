# Plan: Coin Runner v2.2 Development

## 버전 개요

### 목표
v2.1에서 완성된 코어 루프와 레벨 디자인 위에, **신규 유저 첫인상 개선(P0)**, **리텐션·수익화 강화(P1~P2)**, **소셜 기능(P3)**, **Co-op 멀티플레이(P4)**를 단계적으로 쌓아 Roblox 디스커버리 알고리즘에서 경쟁력을 확보한다.

### 핵심 KPI 목표

| 지표 | 현재 추정 | v2.2 목표 | 비고 |
|------|-----------|-----------|------|
| D1 리텐션 | 25~30% | 40%+ | 사운드 + 즉시 재시작 + Zone 1 ActionHint 100% |
| D7 리텐션 | 8~12% | 18%+ | 배틀패스 + 소모형 파워업 + 이름표 확장 |
| D30 리텐션 | 2~4% | 8%+ | 배틀패스 시즌 순환 + 팀 랭킹 + Co-op |
| LTV (90일) | 15~25R$ | 60~80R$ | 고가 번들 + 배틀패스(149R$) + 스타터팩 재설계 |
| 평균 세션 횟수 | 2.5회/일 | 4회/일 | 즉시 재시작 + 소모형 파워업 |
| CCU 배수 | 1x | 2~3x | Co-op Co-Play 신호 (장기) |

### PR #162: 사운드 에셋 적용 — 빈 SoundId 교체 [S]

**목적**: 무음 게임은 6~12세 타겟에게 "미완성" 인상을 준다. 점프, 착지, 피버, 미니피버 사운드를 채워 기본 게임 품질을 확보한다.

**구현 범위**:
- `src/client/Controllers/FeverController.luau` (L206, L214, L223) — 피버 시작음, 종료음, 미니피버음 SoundId 교체
- `src/client/Controllers/PlayerController.luau` (L433, L441) — 점프음, 착지음 SoundId 교체
- Roblox Creator Store에서 무료/유료 사운드 에셋 5종 선정 후 rbxassetid:// 적용

**의존성**: 없음
**예상 작업량**: 1~2h (에셋 탐색 포함)

**완료 기준**:

### PR #163: 게임오버 화면 즉시 재시작 UX 개선 [S]

**목적**: 현재 GameOver 화면에 "PLAY AGAIN" 버튼이 있으나, 로비를 거치지 않고 바로 다음 런을 시작하는 흐름이 5단계(결과→로비→포탈→대기→시작)로 되어 있다면 세션당 플레이 횟수가 30% 감소한다. 기존 RestartButton이 실제로 즉시 런을 시작하는지 검증하고, 서버 라운드트립을 최소화한다.

**구현 범위**:
- `src/client/Controllers/UIController.luau` — RestartButton 클릭 시 `OnRestartButtonClicked()`가 서버에 즉시 GameStart를 요청하는지 확인, 로비 경유 제거
- `src/client/init.client.luau` — GameOver→GameStart 직행 경로 확인/개선
- `src/server/Services/GameManager.luau` — 재시작 시 세션 초기화가 로비 경유 없이 동작하도록 보장

**의존성**: 없음
**예상 작업량**: 2~3h

**완료 기준**:
- [ ] GameOver 화면에서 "PLAY AGAIN" 1탭으로 다음 런 시작 (로비 미경유)
- [ ] 재시작 시 이전 세션 데이터 완전 초기화
- [ ] 부활 가능 상태에서는 부활 버튼이 먼저 표시, 부활 후에야 PLAY AGAIN 노출
- [ ] 결과 화면 → 게임 시작까지 체감 대기시간 2초 이내

**의존성**: 없음
**예상 작업량**: 1~2h

**완료 기준**:
- [ ] Zone 1(Meadow)에서 ActionHint가 100% 발동됨
- [ ] Zone 3(MushroomWood) 이후부터 힌트 빈도가 체감상 줄어듦

### PR #165: 게임 썸네일 + 아이콘 제작 [S]

**목적**: Roblox 디스커버리에서 썸네일과 아이콘이 없으면 CTR이 0에 수렴한다. 게임 내 에셋을 활용한 16:9 썸네일과 512x512 아이콘을 제작한다.

**구현 범위**:
- Roblox Studio에서 게임 씬 스크린샷 캡처 (Zone 1 Meadow 배경 + 캐릭터 + 코인)
- 썸네일: 1920x1080, 밝은 색감, "COIN RUNNER" 텍스트 오버레이
- 아이콘: 512x512, 게임 로고 + 코인 이미지
- Roblox Game Settings > Basic Info에서 업로드

**의존성**: 없음 (코드 작업과 병행)
**예상 작업량**: 2~3h (Studio 작업)

**완료 기준**:
- [ ] 16:9 썸네일이 Game Page에 표시됨
- [ ] 512x512 아이콘이 검색 결과에 표시됨
- [ ] 이미지가 밝고 타겟 연령(6~12세)에 어필하는 색감
