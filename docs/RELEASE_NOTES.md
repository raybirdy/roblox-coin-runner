# Coin Runner 릴리즈 노트

---

## v2.0 — 2026-03-13

> 스토리, 캐릭터, 소셜, 시즌 시스템을 갖춘 대규모 업데이트

### 🆕 신규 기능

#### 스토리 & 챕터 시스템
- **챕터 진행 시스템**: CH1 → CH2 스토리 진행 구조 도입
- **보스 런**: 챕터 클리어 조건 보스 조우 이벤트
- **스토리 자막**: 런 중 실시간 NPC 대화/내레이션 표시
- **Zone Skip Start**: 챕터 진행도에 따라 출발 구역 선택 가능
- **동적 장애물**: 스토리 진행에 따른 난이도 변화

#### 캐릭터 시스템
- **플레이어블 캐릭터 6종**: 러너, 기사, 닌자, 요정, 영웅, 시간여행자
- **캐릭터별 패시브/스킬**: 자석 강화, 피버 연장, 피격 저항, 대시 등
- **캐릭터 상점**: 탭 분리 UI (내 캐릭터 / 🛒 상점), 코인/젬 구매 지원
- **캐릭터 비주얼**: 장착 시 실시간 색상/재질 변경 적용

#### 런 스킬 시스템
- **런 내 XP 획득 + 스킬 선택**: 런 진행 중 레벨업 시 스킬 선택 HUD
- **런 시작 전 스킬 슬롯 2개**: 사전 스킬 장착으로 전략성 강화

#### 소셜 & 랭킹
- **친구 리더보드**: 친구와 점수 비교 탭 추가
- **친구 초대 보상**: 초대 코드 공유 시 양측 보상
- **이름표 시스템**: 색상 3종 + 글꼴 2종 + 프레임 2종 (총 7종 커스터마이징)

#### 콘텐츠 확장
- **시즌 스킨**: 벚꽃 (150젬), 봄 요정 (조각 교환) — 봄 2026 시즌
- **계정 레벨 Lv50**: 레벨 상한 Lv20 → Lv50, 레벨별 칭호 + 특별 보상
- **펫 레벨업 Lv5**: 최대 레벨 상향, 비용 {200, 500, 1200, 3000, 7000}
- **업그레이드 Lv8**: magnet_range, coin_value, lucky_coins 최대 레벨 확장
- **런 목표 리롤**: 마음에 안 드는 런 목표 변경 가능

#### 퀘스트 & NPC
- **로비 퀘스트 NPC**: 포탈 근처 NPC (📜 퀘스트 도우미), ProximityPrompt(E키) → 퀘스트 패널 오픈
- **기사 캐릭터 해금 퀘스트**: 장애물 500개 회피 달성 조건

#### 비주얼 & 연출
- **Zone 비주얼 차별화**: Zone별 안개/장식/장애물 틴트 강화
- **추격자(Chase Mob) 개선**: 비주얼 업그레이드 + 등장 연출
- **이벤트 청크**: 등장 시점 대폭 앞당김

#### 수익화
- **VIP 혜택 확대**: extraLife, freeRevive, 일일 젬, 럭키 스핀 할인
- **개발자 계정 리더보드 제외**: 운영 계정 점수 랭킹 미노출 처리
