| 작업 | 우선순위 | 상태 | 설명 |
|------|----------|------|------|

#### 상품 구성
```
[DevProducts - 젬 구매]
- 젬 50개: 25 Robux
- 젬 200개: 75 Robux (보너스 +50)
- 젬 500개: 149 Robux (보너스 +150)
- 스타터팩: 49 Robux (젬 100 + 랜덤 펫)

[GamePass]
- VIP 패스: 99 Robux (코인 획득 2배 영구)
```
