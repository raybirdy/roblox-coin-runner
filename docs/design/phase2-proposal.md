# Coin Runner Phase 2 기획안

## 개요

v1.0 출시 후 유저 리텐션 강화, 수익화 다변화, 콘텐츠 확장을 위한 Phase 2 제안서.
Roblox 알고리즘 추천 기준 D1 리텐션 12%+ 달성이 1차 목표.

---

## 현재 상태 분석

### 강점
- 탄탄한 코어 루프 (달리기 → 회피 → 코인 → 점수)
- 진행도 시스템 (레벨/미션/업적) 기반 마련
- 수익화 인프라 (젬/VIP/DevProduct) 구축 완료
- 서버 권위 모델 + 보안 검증 완료

### 약점
- **일일 복귀 동기 부족**: 미션은 있지만 로그인 자체에 대한 보상 없음
- **세션 깊이 부족**: 1회 플레이 2~5분, 메타 루프가 "플레이→코인→스킨" 단일 경로
- **소셜 요소 없음**: 완전 싱글 플레이, 친구와 상호작용 불가
- **콘텐츠 고정**: 시즌/이벤트 없이 동일 환경 반복
- **수익화 단일**: 젬 패키지 + VIP만 존재, 광고/구독 미활용

---

## Phase 2 로드맵

### P0: 리텐션 핵심 (출시 후 1~2주 내)

#### 1. 일일 출석 보상 시스템

**목표**: D1 리텐션 +15~25% 향상

```
Day 1: 50 코인
Day 2: 100 코인
Day 3: 5 젬
Day 4: 200 코인
Day 5: 10 젬
Day 6: 300 코인
Day 7: 20 젬 + 한정 트레일
```

- 7일 주기 반복, 연속 출석 시 보너스 배율 (2주차 1.5배, 3주차 2배)
- 출석 끊기면 Day 1부터 리셋 (FOMO 유발)
- 로비 진입 시 팝업 UI 자동 표시

**구현 범위**:
- `PlayerData`에 `loginStreak`, `lastLoginDate` 필드 추가
- `ProgressionService`에 출석 체크 로직 추가
- 클라이언트에 `LoginRewardController` 신규 생성
- 로비 진입 시 보상 팝업 UI

**예상 공수**: 3~4시간

---

#### 2. 럭키 스핀 (데일리 룰렛)

**목표**: 세션 시작 시 흥미 유발, 젬 드립으로 가치 학습

```
보상 테이블:
- 50 코인 (40%)
- 100 코인 (25%)
- 200 코인 (15%)
- 5 젬 (10%)
- 10 젬 (5%)
- 20 젬 (3%)
- 한정 스킨 조각 (2%)  ← 10조각 = 1 한정 스킨
```

- 1일 1회 무료 스핀
- 추가 스핀: 5 젬 또는 광고 시청 (13세 이상)
- 로비에 3D 룰렛 오브젝트 배치

**예상 공수**: 4~5시간

---

#### 3. "신기록 임박" 실시간 알림

**목표**: 세션 연장, "한 판만 더" 유도

- 현재 점수가 최고기록의 80% 도달 시 HUD에 펄스 표시
- `MY BEST: 500 ← YOU: 420` 형태
- 기록 갱신 시 축하 파티클 + 사운드

**예상 공수**: 1~2시간

---

#### 4. 터치 피드백 강화

**목표**: 6~8세 유저의 입력 확신 향상

- 버튼 터치 시 스케일 축소 + 글로우 애니메이션
- HapticService 연동 (점프/슬라이드/코인수집/피격)
- 피격 시 화면 가장자리 빨간 비네트

**예상 공수**: 2~3시간

---

### P1: 수익화 확장 + 소셜 (출시 후 2~4주)

#### 5. 리워드 광고 (Rewarded Video Ads)

**목표**: 비과금 유저 수익화, 무료 보상 경로 제공

Roblox Rewarded Video API 활용 (13세 이상만 노출):

| 사용처 | 보상 |
|--------|------|
| 무료 부활 | 광고 시청 후 추가 부활 1회 |
| 게임 종료 보상 2배 | 광고 시청 후 획득 코인 2배 |
| 추가 럭키 스핀 | 광고 시청 후 스핀 1회 추가 |
| 일일 미션 보상 2배 | 광고 시청 후 미션 보상 2배 |

- 1일 최대 5회 제한
- 13세 미만은 광고 대신 젬 소비 옵션

**구현 범위**:
- `MonetizationService`에 RewardedVideo API 연동
- 게임오버 UI에 "광고 보고 부활" 버튼 추가
- 미션/스핀 UI에 광고 옵션 추가

**예상 공수**: 5~6시간

---

#### 6. 시즌 한정 스킨 & 이벤트

**목표**: FOMO 유발, 정기 복귀 동기, 수익화

```
2주 단위 테마 로테이션:
- 벚꽃 시즌: 핑크 스킨 + 벚꽃 파티클
- 우주 시즌: 갤럭시 스킨 + 저중력 모디파이어
- 할로윈: 호박 스킨 + 다크 배경
- 크리스마스: 산타 스킨 + 눈 이펙트
```

- 스킨 config에 `availableUntil` 타임스탬프 추가
- ShopService에서 기간 만료 체크
- 한정 스킨은 젬 전용 (수익 유도)
- 카운트다운 타이머 표시 ("2일 남음!")

**예상 공수**: 4~5시간

---

#### 7. 친구 챌린지

**목표**: 소셜 바이럴 루프, D7 리텐션

- 로비에서 "친구 도전" 버튼
- 목표 점수 설정 → 친구에게 알림 전송
- 친구가 점수 달성 시 양측 보상 (10 젬)
- Roblox Social API로 친구 목록 조회

**예상 공수**: 5~6시간

---

#### 8. 월간 구독 (Experience Subscription)

**목표**: 안정적 반복 수익

**"Coin Runner Pass"** - 월 99 Robux:

| 혜택 | 내용 |
|------|------|
| 일일 젬 | 매일 5젬 자동 지급 (월 150젬, 패키지 대비 가성비) |
| 구독자 전용 스킨 | 매월 1종 교체 |
| XP 2배 | 레벨업 가속 |
| 광고 없음 | 리워드 광고 대신 직접 보상 수령 |
| 전용 배지 | 닉네임 옆 별 아이콘 |

- Roblox Subscriptions API 활용 (실제 화폐 결제, 자동 갱신)
- `PlayerData`에 `subscriptionActive`, `subscriptionExpiry` 추가

**예상 공수**: 6~8시간

---

### P2: 콘텐츠 확장 (출시 후 1~2개월)

#### 9. 펫 시스템

**목표**: 수집 욕구, 코인 싱크, 장기 목표

```
펫 종류 (6종):
- 고양이 (코인 +5%) - 100 젬
- 강아지 (자석 범위 +10%) - 150 젬
- 토끼 (점프 높이 +5%) - 200 젬
- 용 (보호막 재사용 -10%) - 300 젬
- 유니콘 (XP +10%) - 250 젬
- 피닉스 (부활 속도 +20%) - 400 젬
```

- 펫 레벨업: 코인 투자 → 보너스 증가 (Lv.1~5)
- 게임 중 캐릭터 옆에 작은 펫 따라다님
- Starter Pack에 랜덤 펫 포함 (이미 설계됨)

**예상 공수**: 8~10시간

---

#### 10. Z축 멀티플레이어 (v1.2)

**목표**: 소셜 경쟁, "보여주기" 요소, D7/D30 리텐션

기존 설계(game-design.md)에 따라 구현:

```
레인 배치:
Player 1: Z = 0
Player 2: Z = 30
Player 3: Z = 60
Player 4: Z = 90
```

- 다른 플레이어가 반투명 고스트로 옆에 보임
- 실시간 위치 비교 ("Player2를 추월했습니다!")
- 최종 결과에 순위 표시
- 각자 독립 실행 (서버 부하 최소)

**구현 고려사항**:
- GameManager 멀티세션 동기화
- CameraController FOV 조정 (다른 레인 보이도록)
- 매칭 시스템 (비슷한 레벨끼리)

**예상 공수**: 15~20시간

---

#### 11. 배틀패스 (시즌 패스)

**목표**: D30 리텐션, 장기 목표, 수익화

```
30일 시즌, 30단계:

무료 트랙: 코인, 기본 스킨, XP 부스트
프리미엄 트랙 (99 Robux): 젬, 한정 스킨, 펫, 트레일

진행: 일일 미션 + 게임 플레이로 시즌 XP 획득
```

- 기존 미션 시스템 확장
- 시즌 보상 UI (스크롤 가능한 보상 트랙)
- 시즌 종료 시 미수령 보상 소멸

**예상 공수**: 12~15시간

---

#### 12. 추가 월드 (도시/우주)

**목표**: 콘텐츠 깊이, 장기 플레이 동기

```
월드 해금 조건:
- 초원 (기본): 레벨 1+
- 도시: 레벨 5+ (빌딩 장애물, 자동차, 네온 배경)
- 우주: 레벨 10+ (저중력, 운석, 별 배경)
```

- 각 월드별 고유 청크 템플릿
- 월드별 전용 코인 디자인
- 배경 + 장애물 + BGM 변경

**예상 공수**: 15~20시간 (월드당)

## DataStore 스키마 확장

Phase 2에서 `PlayerData`에 추가되는 필드:

```lua
-- Phase 2 추가 필드
loginStreak: number,              -- 연속 출석 일수
lastLoginDate: string?,           -- "YYYY-MM-DD"
lastSpinDate: string?,            -- 럭키 스핀 마지막 날짜
ownedPets: { string },            -- 보유 펫 ID 목록
equippedPet: string?,             -- 장착 중 펫
petLevels: { [string]: number },  -- 펫별 레벨
subscriptionActive: boolean,      -- 구독 활성 여부
subscriptionExpiry: number?,      -- 구독 만료 시각
seasonPassTier: number,           -- 시즌패스 진행 단계
hasSeasonPass: boolean,           -- 프리미엄 시즌패스 구매 여부
seasonXP: number,                 -- 시즌 경험치
totalAdsWatched: number,          -- 광고 시청 횟수
todayAdsWatched: number,          -- 오늘 광고 시청 횟수
challengesSent: { any }?,         -- 보낸 챌린지 목록
unlockedWorlds: { string },       -- 해금된 월드 목록
skinPieces: { [string]: number }, -- 한정 스킨 조각 수
```

## 수익 예상 모델

**현재 (v1.0)**:
- 젬 패키지 + VIP GamePass

| 수익원 | 예상 기여도 | 비고 |
|--------|-----------|------|
| 젬 패키지 | 30% | 기존 유지 |
| VIP GamePass | 15% | 기존 유지 |
| 리워드 광고 | 20% | 비과금 유저 수익화 |
| 월간 구독 | 20% | 안정적 반복 수익 |
