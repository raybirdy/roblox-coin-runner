## 1. 디자인 철학

### 핵심 원칙

> **"처음 10초 안에 재미를 느끼게 하라"**

| 원칙 | 설명 |
|------|------|
| **쉬운 시작** | 처음 30초는 실패할 수 없을 정도로 쉽게 |
| **점진적 학습** | 새로운 요소는 한 번에 하나씩만 등장 |
| **실패해도 즐거움** | 게임 오버 시에도 보상감을 줌 (획득 코인, 기록 갱신) |
| **시각적 보상** | 코인 수집, 파워업 등에 과장된 이펙트와 사운드 |
| **중독 루프** | "한 판만 더" 심리를 자극하는 짧은 세션 |

### 레퍼런스 분석

| **Tower of Hell (Roblox)** | 청크 랜덤 조합, 재도전 중독성 |

## 2. 월드 비주얼 - "꿈나라 초원" (v1.0)

### 2.1 전체 분위기

- **색감**: 파스텔톤 (연초록, 하늘색, 연분홍, 연노랑)
- **스타일**: 로블록스 블록 느낌 유지하되 둥글둥글한 모서리
- **시간대**: 항상 맑은 낮 (밝고 긍정적인 느낌)
- **하늘**: 뭉게구름이 천천히 흘러감

### 2.2 배경 레이어 (패럴렉스)

```
[Layer 4] 하늘 + 구름       ← 가장 느리게 스크롤
[Layer 3] 먼 산 + 무지개     ← 느리게 스크롤
[Layer 2] 나무 + 꽃밭        ← 중간 속도
[Layer 1] 울타리 + 풀숲      ← 빠르게 스크롤 (캐릭터 바로 뒤)
[Ground]  플레이 영역         ← 캐릭터와 동일 속도
```

- 각 레이어는 서로 다른 속도로 스크롤 → 입체감 연출
- 배경에 간헐적으로 나비, 새가 날아다님 (장식용, 충돌 없음)

#### 패럴렉스 구현 방식
- 각 레이어는 Z축 깊이에 3D 파트로 배치 (Layer4: Z=-80, Layer3: Z=-60, Layer2: Z=-40, Layer1: Z=-20)
- 스크롤 속도 비율: Layer4(10%), Layer3(25%), Layer2(50%), Layer1(75%), Ground(100%)
- **모바일 최적화**: Layer4(하늘) 제거, Layer3을 정적 배경으로 대체 → 3레이어로 축소
- 디바이스 감지: `UserInputService:GetPlatform()` 또는 화면 크기 기반 판별

### 2.3 바닥 타일

| 타일 | 설명 |
|------|------|
| **잔디 블록** | 기본 바닥, 밝은 초록 |
| **흙 블록** | 잔디 아래 레이어 |
| **꽃 블록** | 간헐적으로 꽃이 핀 잔디 (장식) |
| **돌 블록** | 구간 변화 연출용 |

## 3. 청크 시스템

### 3.1 청크 기본 사양

| 항목 | 값 |
|------|-----|
| **청크 길이** | 80 studs |
| **청크 높이** | 40 studs (플레이 가능 영역) |
| **바닥 높이** | 0 studs 기준선 |
| **최대 점프 높이** | 12 studs (2단 점프 기준) |
| **캐릭터 크기** | 약 4×5 studs (폭×높이) |
| **동시 로드 청크** | 3개 (현재 + 앞 2개) |
| **삭제 기준** | 캐릭터 뒤 1청크 거리 초과 시 Destroy |

### 3.2 청크 연결 규칙

```
[청크 A 끝 높이] == [청크 B 시작 높이]
```

- 모든 청크는 시작/끝 높이 태그를 가짐: `LOW(0)`, `MID(6)`, `HIGH(12)`
- 연결 시 **인접 높이만 연결 가능** (1단계 차이까지만 허용):
  - LOW ↔ LOW, LOW ↔ MID: 허용
  - MID ↔ MID, MID ↔ HIGH: 허용
  - LOW ↔ HIGH: **금지** (12 studs 급변 시 낙사 위험)
- 높이 변화 시 경사면 또는 계단을 청크 시작부에 포함

### 3.3 청크 난이도 분류

| 난이도 | 장애물 수 | 코인 밀도 | 특징 |
|--------|-----------|-----------|------|
| **Easy** | 0~1개 | 높음 (코인 잔뜩) | 보상감, 휴식 구간 |
| **Normal** | 2~3개 | 보통 | 적절한 긴장감 |
| **Hard** | 3~4개 | 낮음 (고가치 코인) | 도전 구간 |
| **Bonus** | 0개 | 매우 높음 | 코인만 가득, 보너스 타임 느낌 |
| **NPC** | 0~1개 | 보통 | NPC가 등장하는 특별 청크 |

### 3.4 시간대별 청크 출현 확률

| 구간 (초) | Easy | Normal | Hard | Bonus | NPC |
|-----------|------|--------|------|-------|-----|
| **0~30** | 70% | 10% | 0% | 15% | 5% |
| **30~60** | 40% | 35% | 5% | 10% | 10% |
| **60~120** | 20% | 40% | 20% | 10% | 10% |
| **120~180** | 10% | 35% | 35% | 10% | 10% |
| **180+** | 5% | 30% | 45% | 10% | 10% |

- **첫 30초는 거의 Easy + Bonus** → "나 잘하는데?" 느낌 부여
- Hard 청크는 1분 이후부터 서서히 등장
- Bonus/NPC 청크는 항상 10~15% 유지 → 지루함 방지

## 4. 난이도 곡선

### 4.1 속도 변화

| 구간 (초) | 이동 속도 (studs/s) | 체감 |
|-----------|---------------------|------|
| **0~15** | 25 | 걸어가는 느낌, 여유로움 |
| **15~30** | 30 | 약간 빨라짐, 적응 구간 |
| **30~60** | 38 | 적당한 속도감 |
| **60~120** | 45 | 긴장감 시작 |
| **120~180** | 52 | 꽤 빠름, 집중 필요 |
| **180+** | 55 (최대) | 최고 속도, 고수 영역 |

- 속도는 **EaseOutQuad 곡선**으로 증가 (초반에 빠르게 올라가고 후반에 완만)
  - 공식: `currentSpeed = minSpeed + (maxSpeed - minSpeed) * (1 - (1 - t)^2)` (t = 경과시간/180)
- 최대 속도 캡: 55 studs/s → 이 이상은 반응 불가능, 스트레스만 줌
- 파워업 획득 시 잠시 속도 감소 효과 (현재 속도의 80%, 2초간) → 보상 + 쉬는 타이밍
- 부활 시 속도 80%로 감소 후 5초에 걸쳐 원래 속도로 복귀

### 4.2 장애물 간격

| 구간 (초) | 최소 간격 (studs) | 설명 |
|-----------|-------------------|------|
| **0~30** | 30 | 넉넉한 반응 시간 |
| **30~60** | 22 | 여유 있음 |
| **60~120** | 16 | 적당히 촘촘 |
| **120+** | 12 | 빠른 판단 필요 |

- 장애물 간격 = 플레이어 반응 시간 보장
- 연속 장애물은 최대 3개까지만 (그 이후 반드시 쉬는 구간)

### 4.3 "자비 구간" 규칙

> 아이들의 좌절감을 방지하는 핵심 장치

- 라이프를 잃은 직후 **3초간 무적** + 장애물 없는 구간 보장
- 연속 2회 피격 시 다음 청크는 강제로 Easy 또는 Bonus
- 게임 시작 후 첫 장애물까지 최소 **5초** 보장
- Hard 청크 2개 연속 후에는 반드시 Easy 또는 Bonus 삽입

## 5. 코인 배치 패턴

### 5.1 기본 패턴

```
패턴 1: 직선 라인 (가장 흔함)
○ ○ ○ ○ ○ ○ ○ ○

패턴 2: 아치형 (점프 유도)
        ○ ○ ○
      ○       ○
    ○           ○
──────────────────

패턴 3: 내리막 (슬라이드 후 보상)
○
  ○
    ○
      ○ ○ ○ ○

패턴 4: 지그재그 (위아래 이동 유도)
○       ○       ○
  ○   ○   ○   ○
    ○       ○

패턴 5: 클러스터 (한 곳에 뭉쳐있음)
    ○ ○
  ○ ○ ○ ○
    ○ ○
```

### 5.2 코인 배치 원칙

| 원칙 | 설명 |
|------|------|
| **길잡이 역할** | 코인 라인이 안전한 경로를 알려줌 |
| **점프 유도** | 아치형 코인은 "여기서 점프하라"는 신호 |
| **보상 경로** | 위험한 경로에 고가치 코인 배치 |
| **빈 구간 금지** | 40 studs 이상 코인 없는 구간 금지 (속도 무관한 거리 기준) |
| **리듬감** | 코인 수집이 음악 비트처럼 일정한 리듬을 만듦 |

### 5.3 코인 종류별 배치

| 코인 | 위치 | 배치 방식 |
|------|------|-----------|
| **브론즈** | 모든 곳 | 라인/아치 패턴으로 대량 배치 |
| **실버** | 살짝 높은 곳 | 점프해야 닿는 위치, 3~5개 묶음 |
| **골드** | 장애물 근처 | 위험 보상 - 장애물 바로 위/뒤에 배치 |
| **다이아** | 특별 위치 | NPC 청크 보상, 또는 어려운 경로 끝에 1개 |

## 6. 장애물 상세 설계

### 6.1 장애물별 사양

#### 낮은 벽 (Jump Wall)
```
          ┌────┐
          │    │  높이: 4 studs
──────────┘    └──────────
```
- **회피**: 점프
- **등장 시점**: 게임 시작부터
- **비주얼**: 나무 울타리 / 꽃 화분 더미
- **힌트**: 벽 앞에 코인 아치 패턴 → 자연스럽게 점프 유도

#### 높은 벽 (Slide Tunnel)
```
──────────┐    ┌──────────
          │    │  통로 높이: 3 studs (슬라이드만 통과)
──────────┘    └──────────
```
- **회피**: 슬라이드
- **등장 시점**: 15초 이후
- **비주얼**: 나뭇잎 아치 / 무지개 터널
- **힌트**: 터널 안에 코인 직선 배치 → 슬라이드하면 코인 획득

#### 구덩이 (Gap)
```
──────────      ──────────
          │    │  폭: 6~10 studs
          └────┘
```
- **회피**: 점프
- **등장 시점**: 30초 이후
- **비주얼**: 풀이 없는 흙 구간, 밑에 물 또는 구름
- **힌트**: 구덩이 위에 코인 아치 → 점프 타이밍 안내
- **낙사 판정**: 구덩이 아래로 떨어지면 라이프 -1, 직전 바닥으로 복귀

#### 움직이는 벽 (Moving Wall) - v1.1
```
     ↕ 상하 왕복
  ┌────┐
  │    │  크기: 4×4 studs
  └────┘
──────────────────
```
- **회피**: 타이밍에 맞춰 점프 또는 대기
- **등장 시점**: 90초 이후
- **왕복 범위**: 바닥(0) ~ 8 studs 높이
- **왕복 주기**: 2초 (1초 상승, 1초 하강)
- **통과 윈도우**: 벽이 최고점(8 studs)에 있을 때 아래로 통과 = 약 0.5초
- **비주얼**: 떠다니는 풍선 블록 / 구름 블록
- **힌트**: 움직임 경로에 코인 없음 → "여기는 위험" 신호
- **MVP 제외**: v1.1에서 구현 예정

### 6.2 장애물 조합 규칙

| 규칙 | 설명 |
|------|------|
| 같은 장애물 3연속 금지 | 단조로움 방지 |
| 점프+슬라이드 연속은 간격 충분히 | 조작 전환 시간 확보 (최소 15 studs) |
| 구덩이+높은벽 연속 금지 | 아이들에게 너무 어려움 |
| Hard 청크에서도 해결 불가능한 배치 금지 | 모든 장애물은 반드시 통과 가능해야 함 |

## 7. NPC 시스템

### 7.1 NPC 디자인 원칙

- **귀엽고 둥글둥글한** 비율 (머리가 몸의 1.5배)
- **밝고 따뜻한** 색상 (파스텔톤)
- **항상 웃는 표정** (비위협적)
- **아이들 동작** (뛰기, 손흔들기, 점프)
- NPC는 **장애물이 아님** (부딪혀도 피해 없음)

### 7.2 NPC 종류

#### 응원 NPC (Cheering NPC)
```
위치: 바닥 위, 경로 옆 (뒤쪽 레이어)
행동: 플레이어가 지나가면 환호, 팻말 흔들기
```

##### MVP (v1.0) - 토끼, 곰돌이 2종

| NPC | 외형 | 대사 (말풍선) | 위치 |
|-----|------|---------------|------|
| **토끼** | 분홍 토끼, 큰 귀 | "화이팅!" "멋져!" | Easy 청크 |
| **곰돌이** | 갈색 곰, 꿀단지 | "잘하고 있어!" | Normal 청크 |

##### v1.1 이후 추가

| NPC | 외형 | 대사 (말풍선) | 위치 |
|-----|------|---------------|------|
| **펭귄** | 턱시도 펭귄, 나비넥타이 | "대단해!" | Easy/Normal 청크 |
| **여우** | 주황 여우, 스카프 | "조심해~" | Hard 청크 직전 (경고) |

- 플레이어 접근 시 손/팻말 흔드는 애니메이션
- 말풍선은 1초간 표시 후 사라짐
- 같은 NPC가 연속 등장하지 않도록 랜덤 선택

#### 가이드 NPC (Guide NPC) - 튜토리얼용
```
위치: 게임 시작 구간
행동: 조작법을 알려줌
```

| 순서 | NPC | 말풍선 | 타이밍 |
|------|-----|--------|--------|
| 1 | 부엉이 선생님 | "자동으로 달려! 코인을 모아봐!" | 게임 시작 즉시 |
| 2 | 부엉이 선생님 | "점프! ↑" (+ 점프 버튼 하이라이트) | 첫 낮은 벽 직전 |
| 3 | 부엉이 선생님 | "슬라이드! ↓" (+ 슬라이드 버튼 하이라이트) | 첫 높은 벽 직전 |
| 4 | 부엉이 선생님 | "공중에서 한 번 더 점프!" | 높은 코인 등장 시 |

- **최초 플레이 시에만** 등장 (플래그로 관리)
- 2회차부터는 생략

#### 상인 NPC (Merchant NPC) - 로비 전용 (v1.1)
```
위치: 로비 화면
행동: 상점 NPC 역할 (상점 시스템과 함께 v1.1에서 구현)
```

| NPC | 역할 | 외형 |
|-----|------|------|
| **너구리 상인** | 캐릭터 스킨 판매 | 보따리 진 너구리 |
| **고양이 장인** | 트레일 이펙트 판매 | 안경 쓴 고양이 |
| **강아지 브리더** | 펫 판매 | 모자 쓴 강아지 |

### 7.3 NPC 청크 구성

```
NPC 청크 예시:

[코인라인] [응원NPC] [코인클러스터] [응원NPC] [보너스코인]
                🐰                     🐻
          "화이팅!"               "잘하고 있어!"
──────────────────────────────────────────────────
```

- NPC 청크는 장애물이 적고 코인이 많음 → 보상+힐링 구간
- NPC가 플레이어를 응원하면서 긍정적 감정 유발
- NPC 근처에 고가치 코인 배치 → NPC를 보려고 가까이 감 → 코인 획득

### 9.1 코인 수집 피드백

  - 콤보는 2초간 코인 미수집 시 리셋
  - 콤보 텍스트는 영어지만 크기/색상으로 의미 전달 (읽기 능력 불필요)

## 14. 튜토리얼 플로우

### 최초 플레이 시 자동 튜토리얼

| 단계 | 시점 | 연출 |
|------|------|------|
| 1 | 시작 즉시 | 부엉이 NPC: "안녕! 코인을 모아보자!" + 코인 라인 등장 |
| 2 | 코인 5개 수집 | "잘했어! 코인을 많이 모을수록 좋아!" |
| 3 | 첫 낮은 벽 3 studs 전 | 게임 자동 슬로모션 + "점프해봐!" + 버튼 하이라이트 |
| 4 | 성공 시 | "완벽해!" + 코인 보너스 |
| 5 | 첫 높은 벽 3 studs 전 | 슬로모션 + "슬라이드!" + 버튼 하이라이트 |
| 6 | 성공 시 | "천재인걸?!" + 코인 보너스 |
| 7 | 높은 코인 등장 | "공중에서 한 번 더 점프할 수 있어!" |
| 8 | 30초 경과 | "이제 혼자 할 수 있겠지? 화이팅!" → 튜토리얼 종료 |

- 튜토리얼 중 실패해도 라이프 소모 안 됨
- 2회차 플레이부터 튜토리얼 스킵
- 설정에서 튜토리얼 다시보기 가능

#### 슬로모션 기술 명세
- 튜토리얼 슬로모션: 이동 속도를 현재의 20%로 감소 (클라이언트 로컬 처리)
- 슬로모션 중 청크 로딩/삭제 타이머도 동일 비율로 감속
- 슬로모션 진입/해제는 0.3초 트윈으로 부드럽게 전환
- 튜토리얼 전용 기능이므로 서버 검증 불필요 (클라이언트에서 속도만 조절)

## 15. 청크 제작 가이드 (개발용)

### Roblox Studio에서 청크 제작 시 규칙

```
1. 청크 크기: 80 × 40 studs (가로 × 높이)
2. 원점: 청크 왼쪽 하단 = (0, 0, 0)
3. 바닥 Y=0 기준
4. 높이 태그: 시작/끝 높이를 Attribute로 지정
   - StartHeight: "LOW" | "MID" | "HIGH"
   - EndHeight: "LOW" | "MID" | "HIGH"
5. 난이도 태그: Attribute "Difficulty" = "Easy" | "Normal" | "Hard" | "Bonus" | "NPC"
6. 코인 위치: 빈 Part (투명)로 마킹, 태그 "CoinSpawn" + "CoinType" 속성
7. 장애물: 태그 "Obstacle"
8. NPC 위치: 태그 "NPCSpawn" + "NPCType" 속성
9. 파워업 위치: 태그 "PowerupSpawn" + "PowerupType" 속성
```
