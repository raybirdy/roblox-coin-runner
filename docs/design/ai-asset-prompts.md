# Coin Runner - AI 에셋 생성 프롬프트 가이드

## 개요

Roblox 출시에 필요한 2D 에셋을 AI 이미지 생성 도구로 제작하기 위한 프롬프트 모음.

**대상 도구**: OpenAI (DALL-E / GPT-4o), Claude (이미지 생성), 나노바나나

---

## 1. 게임 아이콘 (512 x 512px)

> Roblox Creator Hub → Game Settings → Icon

### OpenAI (DALL-E 3 / GPT-4o)

```
A vibrant game icon for a Roblox coin runner game.
A cute chibi-style character running to the right, mid-stride with a happy expression.
Shiny gold coins scattered around the character in an arc.
Pastel meadow background with soft green hills and blue sky.
Bold, clean cartoon art style suitable for children ages 6-12.
No text. Square format. Bright saturated colors.
High contrast so it's recognizable at small sizes.
```

### Claude

```
게임 아이콘을 만들어줘 (512x512).
- Roblox 스타일의 귀여운 캐릭터가 오른쪽으로 달리는 모습
- 주변에 빛나는 금색 코인들이 호를 그리며 날아다님
- 배경: 파스텔 톤 초원 (연두색 언덕 + 하늘색 하늘)
- 만화풍, 6~12세 아동 타겟, 밝고 선명한 색감
- 텍스트 없이 캐릭터와 코인만
- 작은 크기에서도 잘 보이도록 높은 대비
```

### 나노바나나

```
[스타일: 카툰/게임아이콘]
Roblox 스타일 코인 러너 게임 아이콘.
귀여운 캐릭터가 달리면서 금화를 모으는 장면.
파스텔 초원 배경, 밝은 색감, 어린이 대상 게임.
정사각형, 텍스트 없음, 고대비.
```

```

The letter "O" in "COIN" is replaced with a shiny gold coin.
Small speed lines behind the text to convey running motion.
Transparent/simple background (solid color or gradient).
Fun, playful typography suitable for a children's game.
The text should be very legible and prominent.
```

### Claude

```
"COIN RUNNER" 게임 로고를 만들어줘.

- 글자 뒤에 속도감을 나타내는 스피드 라인
- 배경은 단색 또는 투명
- 어린이 게임에 어울리는 재미있고 활발한 타이포그래피

```

### 스킨 이미지 적용 방법

1. AI로 각 스킨 이미지 생성 (256x256 권장)
2. https://create.roblox.com → Development Items → Decals → 업로드
3. 각 Decal의 Asset ID 메모
4. `GameConstants.SHOP.SKINS` 테이블에 `thumbnailId` 필드 추가:
   ```luau
   { id = "galaxy", name = "갤럭시", price = 80, currency = "gems",
     color = Color3.fromRGB(60, 20, 120), thumbnailId = "rbxassetid://123456789" },
   ```
5. `ShopController:_createItemFrame`에서 `thumbnailId`가 있으면 ImageLabel로 표시

### 프리미엄 vs 일반 구분 팁

| 구분 | 코인 스킨 | 젬 스킨 (프리미엄) |
|------|-----------|-------------------|
| 배경 | 단순한 단색 글로우 | 화려한 이펙트 + 파티클 |
| 텍스처 | 단색 또는 심플 패턴 | 복잡한 텍스처 (은하, 비늘, 서리 등) |
| 악세사리 | 모자, 망토 등 1개 | 뿔, 왕관 + 이펙트 조합 |
| 전체 느낌 | 귀엽고 친근 | 희소성 있고 특별함 |

## 생성 후 처리 가이드

### 사이즈 조정

| 에셋 | 필요 사이즈 | 비고 |
|------|-------------|------|
| 게임 아이콘 | 512 x 512 px | 정사각형 필수 |
| 게임 썸네일 | 1920 x 1080 px | 최대 3장 등록 가능 |
| Decal (게임 내 이미지) | 자유 | Roblox에 업로드 후 AssetId 사용 |

### 업로드 경로

1. **게임 아이콘/썸네일**:
   - https://create.roblox.com → Creations → 해당 게임 → Settings → Basic Info

2. **게임 내 Decal** (UI 배경 등):
   - https://create.roblox.com → Development Items → Decals → Create
   - 업로드 후 Asset ID를 코드에서 `rbxassetid://` 형태로 사용

### 품질 팁

- **아이콘**: 작은 사이즈에서도 알아보기 쉬워야 함 → 단순하고 대비 높게
- **썸네일**: 게임 검색 결과에서 눈에 띄어야 함 → 밝고 역동적으로
- **아동 타겟**: 무섭거나 폭력적인 요소 배제, 밝고 긍정적인 이미지
- **Roblox 정책**: 저작권 침해 이미지, 부적절한 콘텐츠 업로드 금지

### 생성 시 반복 조정

1. 첫 번째 결과를 확인하고 부족한 부분 피드백
2. 프롬프트에 `"더 밝게"`, `"코인을 더 크게"`, `"배경을 더 단순하게"` 등 추가
3. 여러 변형 생성 후 가장 좋은 것 선택
4. 필요시 Canva, Figma 등에서 텍스트 오버레이 추가
