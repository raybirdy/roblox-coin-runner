# Coin Runner — 시장 조사 & 사업성 분석

**작성일**: 2026-03-17
**버전**: v1.0 (신규)
**대상 게임**: Coin Runner (Roblox 횡스크롤 코인 수집 러너)
**분석 방법**: oracle(시장조사) + prometheus(사업성 분석) + momus(비판적 검토)

---

## 최종 판정

> **CONDITIONAL GO** — 소프트 론치 가능, 단 수익화 파이프라인 완성 후

| 기준 | 판정 | 비고 |
|------|------|------|
| 경쟁사 대비 차별점 | FAIL | Co-op 러너는 가설 미검증 |
| Break-even 18개월 이내 | CONDITIONAL FAIL | 현실적 DAU 2K < BEP 2.9K |
| LTV/CAC > 3:1 | N/A | CAC=0이나 실측 인프라 부재 |
| 팀 도메인 전문성 | PARTIAL PASS | 개발 역량 확인, 퍼블리싱 미검증 |

**소프트 론치 전 필수 완료 항목 (2~4주)**:
1. `BATTLE_PASS.GAME_PASS_ID`, `mega_bundle.productId`, `ultra_bundle.productId` 등록
2. 최소 analytics 구현 (DAU, 과금 이벤트, D1/D7 리텐션)
3. 사운드 에셋 교체 (FeverController/PlayerController SoundId 공란)

## 8. 독점 니치 식별

**타겟 니치**: *"Roblox에서 친구와 함께 하는 Co-op 러너를 원하는 8~14세"*

| 요소 | 내용 |
|------|------|
| 예상 니치 규모 | Roblox Co-op 오비 게임 DAU 합산 ~50만 → 러너 전환 가능 10% = 5만 |
| 확장 경로 | 소셜 러너 니치 → Roblox 러너 전체 → 모바일 러너 Roblox 이주 |
| "비밀" | Roblox는 사교(Socialization) 플랫폼이다. 러너를 Roblox에서 하는 이유는 혼자 할 수 없기 때문이어야 한다. Co-op을 핵심으로 만들면 모바일 러너와 경쟁하지 않아도 된다 |
| 경쟁사가 무시하는 이유 | 공식 IP(Subway Surfers)는 모바일 1위이므로 Roblox Co-op을 최적화할 인센티브가 낮다 |
