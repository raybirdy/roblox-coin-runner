## 목표
Zone별 배경 장식 프롭을 청크에 자동 배치해 세계관 몰입감을 강화한다. IDEA-009로 ProceduralChunkGenerator에 PROP_PALETTE를 추가하고, CanCollide=false 장식 프롭 9종을 경로 양쪽에 배치하여 게임 로직에 영향 없이 시각적 풍성함을 더한다.

## 완료 항목

- 청크당 0~6개 CanCollide=false 장식 프롭 자동 배치
- 경로 양쪽(좌/우) 배치로 주행 경로 미침범
- 게임 로직(충돌, 코인, 장애물) 영향 없음 보장
