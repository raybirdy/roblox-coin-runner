# Coin Runner 수익화 가이드

## 현재 수익 모델 구조

### 1. 듀얼 커런시 시스템

| 통화 | 획득 방법 | 용도 |
|------|----------|------|
| **코인** | 게임 플레이 중 수집 | 기본 스킨 구매 |
| **젬** | Robux 결제 (DevProduct) | 프리미엄 스킨, 시작 파워업 구매 |

### 2. DevProduct (젬 패키지)

| 상품 ID | 젬 | 보너스 젬 | Robux | 비고 |
|---------|-----|----------|-------|------|
| `gems_50` | 50 | 0 | 25 | 기본 패키지 |
| `gems_200` | 200 | +50 | 75 | 인기 패키지 |
| `gems_500` | 500 | +150 | 149 | 대량 패키지 |
| `starter_pack` | 100 | 0 | 49 | 스타터 팩 |

> **배포 전 필수**: Creator Hub에서 DevProduct를 생성하고, `GameConstants.luau`의 `productId = 0`을 실제 ID로 교체해야 합니다.

### 3. VIP GamePass

| 혜택 | 값 |
|------|-----|
| 코인 획득 배율 | 2배 (`coinMultiplier = 2`) |
| 광고 제거 | 향후 |
| VIP 전용 스킨 | 향후 |

> **배포 전 필수**: Creator Hub에서 GamePass를 생성하고, `VIP_GAMEPASS_ID = 0`을 실제 ID로 교체해야 합니다.

## 수익화 전략

### 타겟 유저 프로필
- **연령**: 6~12세
- **결제 주체**: 부모/보호자
- **특성**: 소액 결제 선호, 시각적 보상에 높은 반응

### 가격 전략

#### 저가 진입
- 스타터 팩 (49 Robux ≈ $0.60): 낮은 진입 장벽
- 기본 젬 패키지 (25 Robux ≈ $0.30): 미지불 허들 최소화

#### 대량 구매 유도
- 대량 패키지에 보너스 젬 제공 (gems_500: 30% 보너스)
- 가성비 차이를 명확히 표시

#### VIP 패스
- 일회성 구매로 영구 혜택
- 코인 2배 획득으로 게임 플레이 가치 증대
- 추천 가격: 299~499 Robux

### 콘텐츠 소비 구조

```
[게임 플레이] → [코인 획득] → [기본 스킨 구매]
                                    ↓ (더 희귀한 스킨 원함)
                              [젬 필요] → [Robux 결제]
                                    ↓
                              [프리미엄 스킨 구매]
```

### 스킨 등급별 가격 설계

| 등급 | 통화 | 가격 | 설명 |
|------|------|------|------|
| Common | 코인 | 100~500 | 게임플레이로 획득 가능 |
| Rare | 젬 | 50~100 | 소액 결제 유도 |
| Epic | 젬 | 150~300 | 중간 결제 |
| Legendary | 젬 | 500+ | 프리미엄 |

## 수익 최적화 팁

### 1. 상점 노출 극대화
- 로비에서 상점 NPC 위치를 눈에 띄게 배치
- 게임 오버 화면에서 스킨 미리보기 제공
- 새 스킨 출시 시 알림 배지 표시

### 2. 구매 동기 부여
- 한정 시간 스킨 (FOMO 활용)
- 시즌 테마 스킨 (할로윈, 크리스마스 등)
- 하이스코어 달성 시 스킨 할인 제공

### 3. VIP 가치 강화
- VIP 전용 스킨 2~3개 추가
- VIP 전용 이모트/효과 (향후)
- VIP 전용 리더보드 뱃지

### 4. 무과금 유저 리텐션
- 코인으로 구매 가능한 스킨을 충분히 제공
- 일일 미션 보상으로 젬 소량 지급 (5~10젬/일)
- 연속 출석 보상에 젬 포함

## 광고 수익 모델 (향후 검토)

### Roblox Rewarded Video Ads
- **제한사항**: 13세 이상 유저만 시청 가능
- **요건**: 월 2,000명 이상 방문자
- **현재 판단**: 타겟 연령(6~12세) 불일치로 즉시 도입 부적합

### 도입 시 추천 광고 배치
광고 도입이 가능해지면 아래 시점에 배치:

| 시점 | 보상 | 비고 |
|------|------|------|
| 부활 (무료) | 1회 추가 생명 | 현재 젬 부활 대안 |
| 게임 오버 후 | 코인 2배 보상 | 즉각적 보상감 |
| 로비 일일 보상 | 보너스 코인/젬 | 리텐션 향상 |

### 도입 시기 판단 기준
1. 월 방문자 2,000명 이상 달성
2. 기본 IAP 수익이 안정화된 후
3. 유저 연령층이 13세 이상으로 확장되는 경우

## 주의사항

### Roblox 정책 준수
- 루트박스/가챠 메커니즘 사용 시 확률 공개 필수
- 미성년자 대상 과도한 결제 유도 금지
- DevProduct/GamePass 설명에 제공 내용 명확히 기재

### 밸런스 유지
- Pay-to-Win 요소 지양 (스킨은 외형 전용)
- 무과금으로도 충분히 즐길 수 있는 콘텐츠 보장
- 젬 전용 콘텐츠 비율은 전체의 30% 이하 유지
